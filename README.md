# QMULAutoFindAnswer
## 适用于QMUL的Interactive Video
## 使用方法：
1. 打开视频页面，等待视频加载一会，然后根据你选的方法不同：
   1. 打开开发者工具，粘贴代码到控制台，回车运行   
   2. 打开开发者工具的"源代码"-"片段"，右键运行   
2. 运行之后在标题的左边会出现一个按钮，点击就可以在控制台看到答案。
## 常见问题：
Q: 为什么我运行了没有按钮？   
A: 因为没运行成功，自己检查一下代码有没有粘贴完整，控制台报错是什么   

Q: 点了按钮没反应啊啊啊啊啊啊   
A: 打开控制台再运行一次   

Q: 答案在哪？   
A: 不会用别用
